# スキル：メール分類

## 概要
受信メールを4カテゴリに分類するスキル。Geminiを使った判定と、ルールベース判定を組み合わせる。

## 分類カテゴリ
| カテゴリ | 説明 |
|---|---|
| 要返信（重要） | 期限・依頼・承認など即時対応が必要なもの |
| 要返信（通常） | 返信すべきだが急がないもの |
| 閲覧のみ | 情報共有・CC・ニュースレターなど |
| 無視 | 広告・スパム・自動通知など |

## 判定ロジック
1. 送信者が contacts.md に登録済み → 優先度を参照
2. 件名・本文をGeminiに渡して分類を依頼
3. 信頼度が低い場合はユーザーに確認

## 入力
- メールオブジェクト（件名、送信者、本文冒頭200字）

## 出力
- `{ "category": "要返信（重要）", "reason": "..." }`

## 備考
- 分類結果は学習データとしてMEMORY.mdに蓄積する
