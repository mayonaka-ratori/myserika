# スキル：Telegram 通知

## 概要
処理結果やユーザーへの確認事項をTelegramに送信し、返答を受け取るスキル。

## トリガー
- 新着メールの分類完了時
- 返信案の生成完了時
- エラー発生時

## 通知フォーマット

### 新着メール通知
```
📬 新着メール {n} 件

🔴 要返信（重要）：{n}件
🟡 要返信（通常）：{n}件
📖 閲覧のみ：{n}件

返信案を確認しますか？
[はい] [後で]
```

### 返信案通知
```
✉️ 返信案【{件名}】
宛先：{送信者}

---
{返信案本文}
---

[✅ 承認して送信] [✏️ 修正指示] [❌ 却下]
```

## インタラクション
- ユーザーのボタン押下・テキスト返答を `telegram_bot.py` で受け取る
- `STATE.md` の「承認待ち」欄を更新する

## エラー通知
```
⚠️ エラーが発生しました：{error_message}
```
