# スキル：Gmail チェック

## 概要
Gmailの受信トレイを定期的にポーリングし、未読メールを取得するスキル。

## トリガー
- config.yaml の `check_interval_minutes` ごとに自動実行
- または手動で「メールを確認して」と指示したとき

## 手順
1. `gmail_client.py` の `get_unread_emails()` を呼び出す
2. 取得したメールを `classifier.py` に渡す
3. 分類結果を STATE.md の「未処理メール」に書き込む
4. 要返信メールがあれば `reply-draft.md` スキルへ引き継ぐ

## 出力フォーマット
```
[Gmail-Check] 新着 {n} 件
- 件名：...  送信者：...  分類：要返信（重要）
- 件名：...  送信者：...  分類：閲覧のみ
```

## 備考
- 既読にするタイミングはユーザー承認後とする
- 添付ファイルはファイル名のみ記録（本文は取得しない）
